# Semantic_Search_Engine_for_Excel_Files
Excel is a widely used tool for storing and managing large volumes of data in everyday work. However, extracting relevant information from multiple files can often become tedious and time-consuming, especially when users need to manually search through numerous sheets.

This Search Engine simplifies the process by allowing users to input a query describing the specific information they are looking for. Using a semantic search approach, the engine intelligently analyzes the content and returns the top three sheets that are most relevant to the user’s query.

In addition to this, the system also provides a preview of the content, enabling users to quickly assess its usefulness. Once the desired sheet is identified, users can conveniently download it using the provided download button.

Overall, this solution significantly reduces manual effort, saves time, and improves efficiency when working with large Excel datasets.
